企画から下書きまで、ひとつづき

note記事スタジオ

キーワードを入れると、AIが構成 → 本文下書き → タイトル・ハッシュタグ・SNS告知文まで一気に組み立てます。あなたの体験メモを混ぜて、あなたの記事に仕上げてください。

1 準備→ 2 構成→ 3 本文→ 4 仕上げ
GitHub Pagesやダウンロード後のブラウザでも、下の「AI設定」にご自身のAPIキーを入れればAI生成が動きます。Claude（claude.ai）のプレビュー内で使う場合はキー不要です。noteへの投稿は手動です。AIの下書きは骨組みとして使い、体験と本音を足してから公開するのがおすすめです。【バージョン：v9】
AI設定

AIエンジンをつなぐ

Claude（claude.ai）内で開いている場合：このまま何も設定せず使えます（使えるのはClaudeのみ。セキュリティ制限のため、この画面からGemini・ChatGPTには接続できません）。
GitHub Pagesやダウンロードして使う場合：お使いのAI（Claude／ChatGPT／Gemini）のAPIキーを入力してください。キーはこのページのメモリにだけ置かれ、どこにも保存・送信されません（通信は各AI提供元へ直接行われます）。ページを閉じると消えるので、毎回入力してください。

APIキーの取得方法と注意点（クリックで開く）

■ 最初に知っておくこと：APIキーは、普段のチャット画面の中にはありません
Claude・ChatGPT・Geminiいずれも、いつも会話しているアプリの設定メニューをいくら探してもAPIキーの項目は見つかりません。APIキーは開発者向けの「別のWebサイト」で発行します。ブラウザで新しいタブを開き、下記のアドレスを直接入力して移動してください。

■ Gemini（Google）― まず無料で試したい方に
aistudio.google.com を開きます（チャット版の gemini.google.com とは別サイトです）→ いつものGoogleアカウントでログイン → 「APIキーを取得（Get API key）」→ 発行されたキー（AIza…で始まる文字列）をコピーして、上の入力欄に貼り付けます。カード登録は不要で、無料枠の範囲なら料金はかかりません。最初の一歩におすすめです。

■ Claude（Anthropic）
console.anthropic.com を開きます（チャット版の claude.ai とは別サイト・別アカウントです。途中で platform.claude.com という表示に切り替わっても同じ場所なのでそのまま進めてOK）→「Sign Up」からアカウント作成（メールまたはGoogleアカウント。個人利用なら「Individual」を選択）→ ダッシュボードの「Get API Key」→ 初回はクレジット購入（$5＋税、カード登録）が必要 → キーを作成します。キーは sk-ant- で始まります。キーの全文は発行した瞬間しか表示されないため、その場で必ずコピーして控えてください（閉じてしまったら、削除して作り直せば大丈夫です）。

■ ChatGPT（OpenAI）
platform.openai.com を開きます（チャット版の chatgpt.com とは別サイトです）→ アカウント作成 → クレジットを事前購入（ChatGPT Plusの月額とは別の従量課金です）→「API keys」→「Create new secret key」。キーは sk- で始まります。こちらも発行時にしか全文表示されません。

■ 料金の目安
どの社も使った分だけ課金される従量制です。このツールの用途（1記事あたり構成＋本文で5〜6回の生成）なら、軽量モデル（gpt-4o-mini や 最新のGemini Flash など）で1記事数円以下が目安です。ただし料金は改定されることがあるため、正確な単価は各社の料金ページでご確認ください。

■ 必ず守ってほしい注意点
・APIキーは「あなた名義のクレジットカード」と同じです。他人に教えない、SNSやブログに貼らない、GitHubのファイル内に書き込んで公開しない。
・万一漏れた（漏れたかも）と思ったら、各社の管理画面ですぐにキーを削除（無効化）して、新しく発行し直せば被害を止められます。
・各社の管理画面で利用上限（予算アラート・スペンドリミット）を設定しておくと、想定外の課金を防げます。月数百円程度の低い上限から始めるのが安心です。
・キーはブラウザに記憶させず、パスワード管理アプリやメモ帳（端末内）に保管し、使うときに貼り付ける運用がおすすめです。

GitHub Pagesでの公開手順：GitHubでリポジトリを作成 → このファイルを「index.html」という名前でアップロード → Settings → Pages → Branchを「main」にしてSave → 表示されたURLにアクセス。APIキーをファイル内に書き込んで公開しないでください（URLを知っている人に使われてしまいます）。キーは開いたページの入力欄に都度入れる運用にしてください。

STEP 1

準備する

キーワードは「noteキーワードリサーチ」ツールで見つけた言葉をそのまま使えます。体験メモは箇条書きでOK。ここに書いたことがAIの下書きに織り込まれ、記事の独自性になります。

STEP 2

構成を確認・調整する

AIが提案したタイトルと見出しです。タイトルを1つ選び、見出しや要点は自由に書き換えられます。納得できたら「本文を書く」へ。

STEP 3

本文を整える

AIの下書きです。このまま編集できます。特に体験談の部分は、あなたの言葉で上書きしてください。それが読者の信頼と「スキ」につながります。

STEP 4

仕上げて持ち出す

ハッシュタグとSNS告知文を作り、完成品をコピーまたはファイルで保存します。あとはnoteに貼り付けて、最終チェックをして公開するだけです。
